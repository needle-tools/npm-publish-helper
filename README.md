# Needle Package Publish Helper

Use e.g. in a `package.json` or github action

```json
{
    "scripts": {
      "prepublishOnly": "npx --yes needle-tools/npm-publish-helper",
      "compile": "npx --yes needle-tools/npm-publish-helper compile"
    },
}
```
